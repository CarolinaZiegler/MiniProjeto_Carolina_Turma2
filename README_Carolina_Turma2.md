# Miniprojeto_Carolina_Turma2

#Análise Exploratória de Dados — Base Varejo

Abra o VSCode, navegue até a pasta do projeto e execute o arquivo python análise.py

As Bibliotecas utilizadas foram:
- pandas
- numpy
- matplotlib

O que foi feito no script:
-Carreguei a base Varejo.csv;
-Verifiquei nulos, duplicatas e inconsistências;
-Limpei e padronizei os dados;
-Gerei estatísticas descritivas;
-Criei agrupamentos e gráficos;
-Salvei a base limpa em Base_Varejo_Limpa.csv

Insights obtidos a partir da análise:

-Alimentos representa mais da metade das vendas
totais (384.197 registros), sendo a categoria mais vendida;

-Higiene e Limpeza são o 2º e 3º segmentos mais vendidos, mas bem menos que Alimentos

- O gênero feminino (382.427) compra mais do que masculino (351.020);

- A média de filhos dos clientes é 1,1, com número máximo de 4 filhos;

- 50% dos clientes não têm filhos (mediana = 0);

- Foram encontrados registros com categoria inválida (#N/D), 
tratados como "Sem Categoria";


Conclusões:
-O varejo é dominado por produtos essenciais, sugerindo um perfil de consumo básico/cotidiano;

-Estratégias de marketing podem ser direcionadas ao público feminino, 
que é maioria dos compradores;

-Produtos voltados a famílias pequenas ou sem filhos têm maior potencial nessa base;

-Categorias como Acessórios (12.871) e Bebidas (38.264) têm potencial
de crescimento inexplorado.
